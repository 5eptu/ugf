Promotion and Demotion Process

Promotion Process

Eligibility Criteria for Promotion

  • Consistent Activity
  • Positive Contributions
  • Demonstrated Leadership Qualities
  • Willingness to Assist Others

Steps for Promotion

  1. Review Performance
  2. Prepare Documentation
  3. Use the Command
  4. Notify the Member
  5. Log the Promotion

Demotion Process

Eligibility Criteria for Demotion

  • Inactivity
  • Failure to Meet Responsibilities
  • Violations of Community Guidelines

Steps for Demotion

  1. Evaluate Concerns
  2. Consult with Other Officers
  3. Use the Command
  4. Notify the Member
  5. Log the Demotion

Task Creation Guidelines

Creating Effective Tasks

  • Clearly define the task's purpose and objectives.
  • Specify deadlines for completion.
  • Break down larger tasks into smaller steps.
  • Include resources or links to assist members.

Assigning Tasks

Steps to Assign a Task

  1. Identify the Right Member
  2. Use the Command
  3. Set Expectations
  4. Document the Assignment

Managing Tasks

Tracking Task Progress

Regularly check in with members and address any challenges.

Completing Tasks

Guidelines for Task Completion

  • Submission of Proof
  • Review Process
  • Documentation of Completion

Striking Users

Criteria for Striking

  • Repeated violations of community guidelines.
  • Inappropriate behavior towards other members.
  • Failure to complete assigned tasks after warnings.

Issuing a Strike

Steps to Issue a Strike

  1. Document Incidents
  2. Consult with Other Officers
  3. Use the Command
  4. Notify the Member

Managing Strikes

Record Keeping

Maintain a detailed log of strikes issued.

Removing Strikes

Define a process for removing strikes based on improvement.
